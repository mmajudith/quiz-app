# Quiz App

## Live URL: https://quiz-app-mmajudith.vercel.app/

A web application that allow user to create, edit, delete a quiz, add multiple questions
and options to the quiz and take a quiz.

Technologies Used:

- React
- Reduxtoolkit
- Tailwindcss
- Framer-motion
- React-toastify
- Firebase

